# Irodori Image Asset Instructions

This directory serves Irodori illustration assets to the app.

## Naming

- Preserve official illustration filenames when copying from `docs/Irodori`.
- Group images by level directory, for example `elementary2` or `pre-intermediate`.

## Usage

- Data files reference images with absolute public paths such as `/images/irodori/pre-intermediate/<filename>.png`.
- Add `imageAlt` in pt-BR for learner-facing accessibility.
- Use images when the source activity depends on illustration recognition or visual context.
