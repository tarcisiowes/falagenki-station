# Irodori Source Instructions

This directory contains local source material for Irodori study content.

## Source Of Truth

- Treat PDFs, answer PDFs, MP3 files, and illustration folders here as the source of truth.
- Implement app content faithfully from these files. Preserve lesson order, topics, Can-do goals, grammar points, vocabulary sets, kanji, readings, TIPS, and dialogs.
- Translate English explanations and instructions into Brazilian Portuguese. Do not translate Japanese source text into Japanese variants or simplify it beyond the original intent.

## PDF Workflow

- Use local PDF extraction tools for text, then visually check source pages when layout matters.
- Cross-check extracted text against the PDF before committing content derived from it.
- Keep temporary extraction output outside tracked source directories unless intentionally documenting a reusable process.

## Asset Workflow

- Copy only app-needed audio and images into `public/audio/irodori` and `public/images/irodori`.
- Keep original files in this directory unchanged.
